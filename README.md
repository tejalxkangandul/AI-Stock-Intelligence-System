## ğŸ§  The Working Model

The "brain" of the system is an LSTM architecture specifically chosen for its ability to maintain "memory" of long-term dependencies in time-series data.

### Input Features (Stock DNA)
The model evaluates a 6-dimensional feature vector for every time step:
1. **Close:** The final price of the day.
2. **MA7 & MA21:** Short and medium-term trend indicators.
3. **RSI:** Momentum oscillator to identify overbought/oversold conditions.
4. **Returns:** Daily percentage change.
5. **Volatility:** A standard deviation metric used for both the prediction and the confidence score.

### The Pipeline
1. **Fetch:** Pulls 10+ years of historical data for the selected ticker (e.g., `RELIANCE.NS`).
2. **Process:** Normalizes features and prepares 60-day windows for the LSTM.
3. **Predict:** Outputs the predicted Close price for the next trading day.
4. **Reliability Check:** Calculates a **Confidence Score** by penalizing predictions during periods of extreme market volatility.
